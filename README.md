# Trabalho de Programação Web

O trabalho consistirá no desenvolvimento de uma **aplicação web** dividida em **três fases**, cada uma abordando diferentes aspectos de front-end e boas práticas de desenvolvimento.

## Resumo do Projeto
O sistema é um **site de agendamento online para uma barbearia**, onde o cliente pode visualizar os serviços oferecidos com seus respectivos preços, consultar informações de contato e funcionamento, gerenciar sua conta, e realizar **agendamentos de horário**.  


---

## ✅ Fase 1 – Desenvolvimento da Interface Gráfica
- **Objetivo:** Criar a interface visual da aplicação utilizando **HTML** e **CSS**.
- **Descrição:**  
  Nesta fase, deve ser construída toda a estrutura base da aplicação, definindo a organização dos elementos e aplicando estilos para garantir uma apresentação visual clara e responsiva.
- **Critérios de Avaliação:**  
  - Estruturação correta do HTML.  
  - Uso adequado de CSS (organização, responsividade, design).  
  - Layout funcional e visualmente agradável.

**Status:** [ ✖ ] Concluída  

**Comentários/Observações:**  
> Na primeira fase do projeto, foram implementadas **duas telas principais**: a tela de **login** e a tela **home**.  

> Foram aplicadas as **estilizações CSS** distribuídas nos arquivos `common.css`, `login.css` e `home.css`, garantindo consistência visual, **bom design** e **responsividade** em diferentes tamanhos de tela.  

> Além disso, foram utilizados **pequenos scripts em JavaScript** para melhorar a navegabilidade e demonstrar recursos adicionais da página. Esses scripts, por serem simples, estão **incorporados diretamente nos arquivos HTML** de forma provisória, servindo como base para funcionalidades mais completas nas próximas fases.  

> As páginas apresentam **layout organizado**, elementos interativos básicos e estrutura lógica que **facilitam a expansão futura da aplicação**, servindo como base sólida para a inclusão de funcionalidades mais avançadas.

## ✅ Fase 3 – Cookies, Armazenamento Local e Padrões de Desenvolvimento
